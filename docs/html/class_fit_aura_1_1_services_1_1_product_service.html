Detailed Description

Serwis obsługujący zapytania do API w celu wyszukiwania i pobierania danych o produktach spożywczych.

Odpowiada za pobieranie szczegółowych informacji o produktach na podstawie kodów kreskowych oraz wyszukiwanie produktów w bazie danych według frazy.
