Detailed Description

Serwis przechowujący stan bieżącego.

Klasa wykorzystuje wzorzec obserwatora. Komponenty UI powinny subskrybować zdarzenie OnChange, aby reagować na aktualizacje danych w czasie rzeczywistym.
