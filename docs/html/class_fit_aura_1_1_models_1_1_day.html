Detailed Description

Reprezentuje dzienny zapis aktywności, odżywiania i pomiarów użytkownika.

Klasa pełni rolę modelu danych, który agreguje informacje o zdrowiu i aktywności użytkownika w ramach konkretnej daty kalendarzowej.
